Why Archive?

Web citations decay over time. ArchiveLinks helps journals preserve cited pages before publication so readers can access evidence even when original URLs change or disappear.

  • Klein et al. (2014), Scholarly Context Adrift, PLOS ONE: https://doi.org/10.1371/journal.pone.0115253
  • Internet Archive, Wayback Machine: https://web.archive.org/

Privacy Model

  • Manuscript parsing is done locally in your browser.
  • The manuscript file is not uploaded to an ArchiveLinks server.
  • Only extracted URLs are sent to Archive.org for preservation requests.

Workflow

  1. Upload a PDF or DOCX file using drag-and-drop or file picker.
  2. Review extracted links, delete incorrect ones, and add missing URLs manually.
  3. Use bulk selection to archive selected links, then retry unresolved links.
  4. Export a CSV with original links, archive links, and statuses.
